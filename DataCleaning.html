Data Cleaning in SQL Server

In this project we take raw housing data and transform it in SQL Server to make it more usable for analysis

SQL Cleaning Process:

  • Standarized date format: The original data had the SalesDate column as DateTime type and we needed as Date type
  • Populate Property Address data: We had some null values, so analysing, the ParcelId had the address information we needed
  • Breaking out Address and OwnerAddress into individual columns: For better data management, we decided to split the Address column into separate columns to have the address and city separated.
  • Remove duplicates: We used a window function to remove the duplicate values.
  • Delete unused columns: We removed some columns that we considered would not be used to optimize data management
