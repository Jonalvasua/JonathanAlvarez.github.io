Amazon Web Scrapping with Python

In this project we scrape data from Amazon to analyze price data for a product

Jupyter Notebook:

  • Labraries import: We imported the different libraries that we are going to use
  • Connect to website: We conected the Amazon URL
  • Assign where to find: We looked where is the information we wanted (price, title)
  • Trim characters: We trim the spaces and the $ sign
  • Append data to cvs: We created a csv file and append the data
  • Automation of the price tracker: Automate the extraction of the data to be daily
  • Extra: If the price drops below a certain amount, an email will be sent to notify about the discount.
