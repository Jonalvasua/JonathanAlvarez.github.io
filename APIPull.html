Crypto API Pull with Python

In this project we look at automating this API pull as well as cleaning up the data a little bit. Then we create a visualization or two just to be safe in Jupyter Notebooks

Jupyter Notebook:

  • API Pull: We extracted the data from the API of CoinMarketCap
  • Impor Pandas: We imported the library to visualize the data and create column with timestamp
  • Automating pull: Created an automated API pull and append it into a .csv file
  • Import csv file: We imported the csv file and cleaned a little bit
  • Seaborn library: Imported the library for few visualizations
